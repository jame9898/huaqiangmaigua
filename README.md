# 🍉 敲瓜辨生熟

> 掏出手机敲两下，用频率数据辅助判断西瓜生熟。

一个纯前端的西瓜声学检测工具。用手机麦克风采集敲击声，本地 FFT 频谱分析算出主频率，结合多次测量取平均判断生熟。**音频全程本地处理，不上传任何服务器**，部署后手机添加到主屏幕就像原生 App，户外买瓜离线也能用。

---

## 📖 频率原理

西瓜被敲击后会产生振动，振动的**主频率（基频）**和成熟度相关：西瓜成熟后瓜瓤弹性模量变小，对质量一定的西瓜而言固有频率降低，因此通过频率可以估算成熟度。

### 🎯 生熟判断阈值（简化参考）

| 平均主频率 | 参考判断 | 含义 |
|-----------|---------|------|
| **> 189 Hz** | 🟢 生瓜（偏生） | 瓜瓤致密、含水量高，振动频率高，"梆梆"响 |
| **133 – 189 Hz** | 🟡 熟瓜（适中） | 瓜瓤糖分转化、组织软化，振动频率适中，"嘭嘭"响 |
| **< 133 Hz** | 🔴 可能过熟/空心 | 瓜瓤过软或空心，振动频率低，"噗噗"闷响 |

### 📚 数据来源与简化说明

本工具的判断思路参考自**《农业工程学报》**公开发表的研究：

> 王书茂, 焦群英, 籍俊杰. 西瓜成熟度无损检验的冲击振动方法[J]. 农业工程学报, 1999, 15(3): 240-245.
> （作者单位：中国农业大学、河北省农业机械化研究所）
> 原文链接：http://www.tcsae.org/cn/article/id/19990350

需要说明的是：
- **原研究使用的是与西瓜质量相关的成熟度指数 f²m^(2/3)**（频率与质量综合判定），并非单一固定频率阈值。
- 本工具为简化易用，**将其简化为固定频率阈值**（133/189Hz），未引入质量修正，因此精度有限。
- 该论文发表于 1999 年，研究品种为"京欣"西瓜，不同品种、产地、个体的实际频率范围可能存在差异。
- **若你需要更准确的数据，欢迎有相关研究或实测数据的朋友贡献补充**，共同完善参考标准。
- **若使用者在实测中发现数据与判断结果存在明显差异，欢迎联系我并上报对应数据**（品种、质量、实测频率、实际生熟情况），以便持续校准阈值。

## 🚀 使用方法

### 手机用户（推荐）
1. 用 **Chrome 浏览器**（别用微信内置浏览器）访问在线网址：
   ```
   https://jame9898.github.io/huaqiangmaigua/
   ```
2. 选品种 → 点"开始测量"→ 允许麦克风 → 倒计时内用**指关节轻敲瓜身中部（腹部）**→ 连续 3 次
3. 看平均频率和判断结果，点"保存记录"存入历史
4. **添加到主屏幕**：Chrome 菜单 ⋮ → "添加到主屏幕" → 桌面生成 🍉 图标，以后点图标全屏打开、离线可用
