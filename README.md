# 🍉 敲瓜辨生熟 · 华强买瓜神器

> **"这瓜保熟吗？"**
> —— 现在不用瞪眼看了，掏出手机敲两下，让科学替华强回答这个灵魂拷问。

一个纯前端的西瓜声学检测工具。用手机麦克风采集敲击声，本地 FFT 频谱分析算出主频率，结合多次测量取平均判断生熟。**音频全程本地处理，不上传任何服务器**，部署后手机添加到主屏幕就像原生 App，户外买瓜离线也能用。

## 📖 这瓜到底保不保熟？—— 频率原理

西瓜被敲击后会产生振动，振动的**主频率（基频）**和成熟度直接相关。这和拍西瓜听声音是同一个道理，只不过我们把"耳朵听"换成了"麦克风+FFT 量化"。

### 🎯 生熟判断阈值（通用参考）

| 平均主频率 | 参考判断 | 含义 |
|-----------|---------|------|
| **> 189 Hz** | 🟢 生瓜（偏生） | 瓜瓤致密、含水量高，振动频率高，"梆梆"响 |
| **133 – 189 Hz** | 🟡 熟瓜（适中） | 瓜瓤糖分转化、组织软化，振动频率适中，"嘭嘭"响 |
| **< 133 Hz** | 🔴 可能过熟/空心 | 瓜瓤过软或空心，振动频率低，"噗噗"闷响 |

> ⚠️ 以上为**通用参考阈值**，不同品种（8424、麒麟、甘美、特小凤、黑美人等）有差异。本工具给出的是参考性判断，不构成品质保证——毕竟华强也不是每次都看得准。

## 🚀 使用方法

### 手机用户（推荐）
1. 用 **Chrome 浏览器**（别用微信内置浏览器）访问在线网址：
   - 国内推荐：`https://linhanduzikai.gitee.io/huaqiangmaigua/`
   - 备用：`https://jame9898.github.io/huaqiangmaigua/`
2. 选品种 → 点"开始测量"→ 允许麦克风 → 倒计时内用**指关节轻敲瓜身中部（腹部）**→ 连续 3 次
3. 看平均频率和判断结果，点"保存记录"存入历史
4. **添加到主屏幕**：Chrome 菜单 ⋮ → "添加到主屏幕" → 桌面生成 🍉 图标，以后点图标全屏打开、离线可用
